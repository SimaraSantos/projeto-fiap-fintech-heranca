# Projeto FinTech - FIAP 💰

Este é um sistema de gerenciamento financeiro pessoal desenvolvido como parte do curso de **Análise e Desenvolvimento de
Sistemas na FIAP**.

## 🚀 Funcionalidades

- Cadastro de transações (Gastos e Investimentos).
- Cálculo de saldo e rendimentos.
- Menu interativo via console.
- Uso de conceitos de POO (Herança, Encapsulamento e Polimorfismo).

## 🛠️ Tecnologias Utilizadas

- **Java 17** (ou a versão que você estiver usando).
- **Git & GitHub** para controle de versão.
- **IntelliJ IDEA** como IDE.
